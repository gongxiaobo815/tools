Querying the Mind Studio Version

Querying the Mind Studio Version

You can query the version by running a command in the background or using the GUI.

  • Query the version by running a command in the background

    After Mind Studio is installed, go to the ~/tools/conf directory and run the cat version command to view the Mind Studio version, as shown in Figure 1.

    Figure 1 Querying the Mind Studio version in the background

  • Query the version on the GUI

    Log in to Mind Studio and choose Help > About. The Mind Studio version information is displayed, as shown in Figure 2.

    Figure 2 Querying the Mind Studio version in the foreground

NOTE:
The time information shown in Figure 1 and Figure 2 is for reference only.
