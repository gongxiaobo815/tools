Macro: HIAI_REGISTER_SERIALIZE_FUNC

Macro: HIAI_REGISTER_SERIALIZE_FUNC

Provides user-defined serialization and deserialization mechanisms for user-defined data types. This macro is defined in data_type_reg.h.

That is, when data is sent, the structure pointer transferred by the user is converted into the structure buffer and data buffer. When data is received, the structure body buffer and data buffer obtained from the framework are converted back into the structure.

Application scenario:

This API is used to quickly move data from the host to the device. If you need to improve the transmission performance, you must use this registration function.

NOTE:

  • This API is used to quickly move data from the host to the device.
  • If you want to use a high-performance transmission interface, you must register the serialization and deserialization functions with this registration function.

Syntax

HIAI_REGISTER_SERIALIZE_FUNC（name, type, hiaiSerializeFunc, hiaiDeSerializeFunc）

Parameter Description

Parameter

Description

Value Range

name

Name of a registered message

-

type

Type of user-defined data

-

hiaiSerializeFunc

Serialization function. The function is in the following format:

typedef void(*hiaiSerializeFunc)(void* inputPtr, std::string& ctrlStr,

uint8_t*& dataPtr, uint32_t& dataLen);

The parameters are described as follows:

  • inputPtr: structure pointer
  • ctrlStr: structure buffer
  • dataPtr: buffer of the structure data pointer
  • dataLen: structure data size

-

hiaiDeSerializeFunc

Deserialization function. The function is in the following format:

typedef std::shared_ptr<void>(*hiaiDeSerializeFunc)(

const char* ctrlPtr, const uint32_t& ctrlLen,

const uint8_t* dataPtr, const uint32_t& dataLen);

The parameters are described as follows:

  • ctrlPtr: structure pointer
  • ctrlLen: control information size in the data structure
  • dataPtr: structure data pointer
  • dataLen: size of the data storage space in the structure. This parameter is used only for verification and does not indicate the size of the original data.

-

Return Value

For details about the returned error codes, see "Error Codes."

Error Codes

No.

Error Code

Description

1

HIAI_OK

The running is OK.

2

HIAI_ENGINE_FUNCTOR_NULL

The HiAI Engine function is null.

3

HIAI_ENGINE_FUNCTOR_EXIST

The HiAI Engine function exists.
