Creating a Mind Project

Creating a Mind Project

Skip this section if you have developed an operator in the operator development project GUI, which is directed from the model conversion failure GUI. That is, the Mind project has been created in Creating an Operator Development Project After a Conversion Failure. To create a Mind project, perform the following steps:

  1. Choose File >New > New Project > ....

  2. In the New Project dialog box that is displayed, choose Mind Engine Project > Mind Project. The configuration window is displayed, as shown in Figure 1.

    Figure 1 Creating a Mind Engine project

    Configure the project by referring to Table 1.

    Table 1 Parameter description

    Parameter

    Description

    Name

    Indicates the project name, which is configured as required.

    A name must be a character string without spaces or Chinese characters. A space is automatically filled with a -.

    Mind Type

    Indicates the project type. DEFAULT indicates that the Mind Studio orchestration function is used.

    Target
    • ASIC indicates that the running target is the EVB or PCIe board.
    • Atlas DK indicates that the running target is a developer board.

    Source Code From

    Empty: The project is self-developed and is not imported from external systems.

  3. Click Create to create a Mind project. A mind file with the same name as the project name is automatically generated, as shown in Figure 2. The file cannot be copied, deleted, or renamed.

    Figure 2 Creating a Mind project
